# CF Data Workshop HA-1

Проект содержит набор данных по публичным российским компаниям (MOEX). Данные объединяют рыночную информацию, фундаментальные показатели, бухгалтерскую отчётность и альтернативный источник в виде новостей компаний.

## Источники данных
- **T-Invest API** - рыночные данные (дневные цены и объёмы) и фундаментальные мультипликаторы.
- **СПАРК** - бухгалтерская финансовая отчётность компаний за 2024 год.
- **Finam** - альтернативный источник: новости и публикации по компаниям.

## Назначение файлов и папок
- **`main.ipynb`** - основной ноутбук с полным пайплайном сбора, объединения и очистки данных.
- **`final_dataset.xlsx`** - финальная, очищенная версия датасета для сдачи и анализа.
- **`parsed_news/`** - новости компаний с Finam (по одной CSV на тикер).
- **`stocks/`** - котировки акций за последние 3 года.
- **`csv/`** - промежуточные таблицы.
